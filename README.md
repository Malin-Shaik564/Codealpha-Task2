# Unemployment Analysis with Python

## Project Overview
This project focuses on analyzing unemployment rate data using Python to understand unemployment trends and identify the impact of Covid-19 on employment levels.

The analysis includes data cleaning, visualization, and trend analysis to generate meaningful insights for economic and social decision-making.

---

## Objective
- Load and analyze unemployment rate data.
- Clean and preprocess the dataset.
- Visualize unemployment trends over time.
- Study the impact of Covid-19 on unemployment.
- Identify important patterns and seasonal trends.
- Generate insights for policy and planning.

---

## Dataset Information
The dataset contains:
- Region/State information
- Date/Time period
- Estimated Unemployment Rate (%)
- Labour Participation Rate
- Employment estimates

Dataset source: Unemployment in India Dataset

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Steps Performed

### 1. Import Libraries
Imported required Python libraries for data analysis and visualization.

### 2. Load Dataset
Loaded the unemployment dataset using Pandas.

### 3. Data Cleaning
Performed:
- Handling missing values
- Removing duplicate records
- Formatting date columns
- Checking data types

### 4. Exploratory Data Analysis (EDA)
Analyzed:
- Dataset shape
- Summary statistics
- Correlation between variables

### 5. Data Visualization
Created:
- Line charts for unemployment trends
- Bar charts by region/state
- Heatmaps for correlation
- Covid-19 impact comparison charts

### 6. Insights Generation
Identified:
- Highest unemployment periods
- Regional differences
- Covid-related unemployment spikes
- Seasonal patterns
